# Power BI Pricing & Profitability Analytics

## Project Overview
This project showcases an end-to-end Power BI solution focused on pricing strategy, profitability analysis, and business-driven insights.

The dashboard simulates real-world decision-making scenarios, allowing management to understand how price changes impact revenue and net profit across products and sales channels.

## Key Features
- Executive KPI overview (Revenue, Net Profit, Margin, Orders)
- Product performance analysis (Top & Bottom products by revenue)
- Pricing What-If analysis using Power BI parameters
- Profit sensitivity to price changes
- Channel-level profitability impact

## Dashboard Pages
### 1. Executive Overview
High-level KPIs and revenue/profit trends.

### 2. Product Performance
Identification of top-performing and underperforming products by revenue.

### 3. Pricing & What-If Analysis
Simulation of price changes and their impact on revenue and net profit, including sensitivity analysis.

## Tech Stack
- Power BI
- DAX (advanced measures & What-If parameters)
- CSV-based star schema model

## How to Use
1. Download the `.pbix` file from the dashboard folder
2. Open it in Power BI Desktop
3. Data is included locally via CSV files (no external connections)

## Business Value
This dashboard supports pricing decisions by highlighting how small price adjustments can disproportionately affect profitability, especially across different sales channels.
